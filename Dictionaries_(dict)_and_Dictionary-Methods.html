Python Dictionaries

What is a Dictionary?

A dictionary in Python is an unordered collection of key-value pairs. Keys must be unique and immutable.

What is a Dictionary in Python?

A dictionary stores data in key-value pairs.

  • It is mutable (can be changed).
  • Keys must be unique and immutable (strings, numbers, tuples).
  • Values can be of any type (string, number, list, dictionary, etc.).
  • Dictionaries are created using {} (curly braces).

Dictionary Methods

  • dict.keys() - Returns all keys
  • dict.values() - Returns all values
  • dict.items() - Returns all key-value pairs
  • dict.update(new_dict) - Updates dictionary
  • dict.clear() - Clears all items
  • dict.popitem() - Removes the last inserted key-value pair
  • dict.update(new_dict) - Updates dictionary with another dictionary
  • dict.copy()- Returns a shallow copy of the dictionary

Creating a Dictionary

student = {"name": "John", "age": 25, "course": "Python", "marks": 90}
print(student)

Accessing Dictionary Values

print(student["name"])  # Output: John
print(student.get("age"))  # Output: 25

Modifying a Dictionary

student["marks"] = 95  # Adding new key-value pair
student["age"] = 26  # Updating existing key
print(student)

Removing Items

student.pop("age")  # Removes the key 'age'
print(student)

Looping Through a Dictionary

for key, value in student.items():
print(f"{key}: {value}")

Nested Dictionaries

students = {
"student1": {"name": "Alice", "age": 22, "course": "Python"},
"student2": {"name": "Bob", "age": 24, "course": "Java"}
}
print(students["student1"]["name"])  # Output: Alice

Real-World Example: Employee Database

employees = {
101: {"name": "Alice", "age": 30, "department": "HR"},
102: {"name": "Bob", "age": 25, "department": "IT"},
103: {"name": "Charlie", "age": 28, "department": "Finance"}
}
emp_id = 102
if emp_id in employees:
print(f"Employee Name: {employees[emp_id]['name']}")
else:
print("Employee not found")
