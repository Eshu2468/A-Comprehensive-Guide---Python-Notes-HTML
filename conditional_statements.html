Python Conditional Statements

1️. if Statement

The if statement executes a block of code if a condition is true.


age = 20
if age >= 18:
print("You are eligible to vote.")

2️. if-else Statement

The if-else statement executes one block if true, another if false.


age = 16
if age >= 18:
print("You are eligible to vote.")
else:
print("You are not eligible to vote.")

3️. if-elif-else Statement

The if-elif-else statement checks multiple conditions.


marks = 85
if marks >= 90:
print("Grade: A")
elif marks >= 75:
print("Grade: B")
elif marks >= 60:
print("Grade: C")
else:
print("Grade: F")

4️. Nested if Statements


age = 25
is_registered = True

if age >= 18:
if is_registered:
    print("You can vote.")
else:
    print("You need to register first.")
else:
print("You are too young to vote.")

5️. Logical Operators

Using and Operator


age = 20
has_voter_id = True

if age >= 18 and has_voter_id:
print("You can vote.")

Using or Operator


is_raining = False
has_umbrella = True

if is_raining or has_umbrella:
print("You are prepared!")

Using not Operator


is_sunny = False

if not is_sunny:
print("It is not sunny today.")

6. Short-Hand if and if-else


# One-line if
age = 20
if age >= 18: print("Eligible to vote.")

# One-line if-else (Ternary Operator)
age = 16
print("Eligible to vote.") if age >= 18 else print("Not eligible to vote.")

Using if in a Loop

Print even numbers from 1 to 10.


for num in range(1, 11):
if num % 2 == 0:
    print(num)

Shorts notes

  1. if: Checks a condition and executes code if True.
  2. else: Executes alternative code when the if condition is False.
  3. if-else: Combines if and else for two possible cases.
  4. if-elif-else: Handles multiple conditions.
  5. nested if: An if inside another if.
  6. Short-hand if and if-else: Write conditions in a single line.
